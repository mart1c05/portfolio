PROJECTS

Free Britney Spil (Animation)

In this course we learned about characterdesign and Javascript. Our task was to create the characters and gamedesign in Illustrator. The finished game had to be developed using HTML, CSS and Javascript.

Unesco's Verdensmålsskolerne

This is a caseproject, solved in groups of 5. Our case was to come up with a better solution for "Unesco's verdensmålsskoler" information archetecture. The solution had to be built in Wordpress (CMS) using PHP files. We especially had to focus on creating a more user-friendly filtration system.

EACE gum (2. semester examproject)

For our examproject we worked in a group of 5. We had to find a start-up company and help them redesign their website, produce a video and come up with marketing/SoMe strategies that would boost the company. The ending product was built in Wordpress (CMS) through a childtheme, but developed from the bottom in PHP files using HTML, CSS and Javascript.
